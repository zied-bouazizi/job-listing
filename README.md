## **JobsPortal**

Jobs portal application with Laravel 9 and MySQL

**Requirements**

Check Laravel 9 Requirements: https://laravel.com/docs/9.x

**Installation**

-   Install Dependencies
```sh
composer install
```
-   Set Database Credentials & App Settings in `.env` file

-   Migrate Database
```sh
php artisan migrate
```
-   Database Seed
```sh
php artisan db:seed
```
